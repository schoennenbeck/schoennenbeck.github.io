In this table you can find MAGMA-readable files (simply download and use the "load"-command) containing our results for some S-arithmetic groups of exceptional groups. Note that these groups are not S-unit groups of orders. The files generally contain a finitely presented group G as well as a list X corresponding to the generators of G (in some representation of the underlying arithmetic group).
